# 1주차 과제 안내

## 📌 과제 제출 안내
각자 자신이 푼 코드를 입력하여 해당 폴더에 올려주시면 됩니다.  
문제 파일은 미리 준비해두었으니, 확인 후 제출해 주세요.

---

## 💻 제출 방법
1. **VSCode 사용 시**  
   - **Extension**에서 `BOJ Tester`를 다운받아 주세요.
   - 파일을 열고, '문제 보기' 버튼을 클릭하면 문제를 바로 확인할 수 있습니다.

2. **Git 작업**  
   - 코드를 업로드하기 전에 반드시 `git pull`을 먼저 진행해 주세요.  
   - 커밋 메시지는 자유롭게 작성해주세요. 예시:  
     `feat: 문제 1번 풀이 추가`

---

## ⚠️ 유의 사항
- 각자의 코드가 담긴 파일을 올려주세요.
- 코드 업로드 전 꼭 `git pull`을 해주세요.
- 코드 제출 시 문제와 관련된 모든 파일을 정확히 업로드해야 합니다.
- 제출 기한을 지켜주세요.
